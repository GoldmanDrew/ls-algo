Sign in

Paste a GitHub fine-grained personal access token (PAT) with read access to the private repo. The token never leaves your browser; it is stored only in this tab's sessionStorage and discarded on logout.

  1. Open github.com / Settings / Developer settings / Fine-grained tokens and create a new token.
  2. Resource owner: your account. Repository access: only the ls-algo repo.
  3. Permissions: Repository » Contents = Read-only; Metadata = Read-only (always).
  4. Set a short expiration (7 / 30 days). Copy the token. Paste below.

The static page itself is harmless. All numbers shown live in the private ls-algo repo and are fetched only after GitHub validates that you have read access. If you are not a collaborator, sign-in fails.
